## 🔖 2. 프로젝트 소개
- 소요시간: 2주
- 참여 인원: 총 4명
### 2.1. 목표
- 엔터 쇼핑몰을 모티브로 한 웹사이트 구현

### 2.2. 담당 구현사항
### 2.2.1. **Frontend**
- 상단이미지 우측 무한슬라이드 구현
- 동물, 상품, 가격 등 **sorting** 된 상품들 보여지게 동적 구현
- 6개 기준으로 더보기 **pagination** 구현

### 2.2.2. **Backend**
- **MVC Pattern**으로 카트 **CRUD API** 구현
- **Prisma**를 이용한 **Mysql Database** 구현
- **AWS(EC2 & RDS)**를 통해 서비스 배포 경험

### 2.3. 기술 스택
- react
- scss
- Prisma
- MySQL

## 💡 4. 개발후기

### 4.1. 얻게된 점
- scss의 사용법
